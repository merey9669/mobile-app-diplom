Полнофункциональное мобильное приложение для управления сотрудниками на базе Expo, React Native и Firebase.

## 🎯 Для дипломной работы

Это приложение создано специально для дипломной работы и включает:
- ✅ Полную интеграцию с Firebase (Authentication, Firestore)
- ✅ Регистрацию и авторизацию пользователей
- ✅ Real-time синхронизацию данных
- ✅ Два типа ролей (Сотрудник / Руководитель)
- ✅ Все CRUD операции
- ✅ Современный UI/UX дизайн
- ✅ Готовое к использованию
- ✅ Использует только бесплатный тариф Firebase

## Функционал

### Основные модули:
- **Splash Screen** - Экран загрузки с анимацией
- **Onboarding** - Знакомство с приложением (3 экрана)
- **Авторизация** - Вход как сотрудник или руководитель
- **Новости** - Корпоративная лента новостей
- **Чек-листы** - Списки задач с прогрессом
- **Задачи** - Система управления задачами
- **Чат** - Корпоративный мессенджер
- **Документы** - Доступ к документам
- **Магазин** - Каталог товаров за баллы

### Возможности:
- Разные роли (Сотрудник / Руководитель)
- CRUD операции для новостей, задач, чек-листов
- Отслеживание статуса задач
- История изменений
- Обмен сообщениями в чате
